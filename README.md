# Ottomatik Webhook to Slack via IBM Cloud functions

tl;dr:

1. Get your Slack incoming webhook URL
2. `npm install`
3. `composer install`
4. `sls deploy` |
5. Copy the web action endpoint and place that into Ottomatik
